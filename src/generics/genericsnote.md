generic = membuat komponen (fungsi, interface, type) agar bisa bekerja dengan berbagai macam tipe data namun tetap menjaga keakuratannya.
Bisa juga untuk mencegah developer menggunakan tipe data any

generics juga bisa digunakan di type serta interface untuk struktur data yang konsisten namun bervariasi

generics constraint = membatasi tipe data yang bisa masuk

///////////////////////////////////////////////

Gunakan generics jika ingin mempertahankan hubungan tipe antar parameter dan hasil, atau antar beberapa properti
- ketika memiliki fungsi yang bekerja dengan berbagai tipe data input dan ingin mengembalikan tipe yang berkaitan dengan input tersebut
- untuk membuat utility function yang bisa bekerja di berbagai modul
